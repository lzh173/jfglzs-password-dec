## 用法
打开release里释放的二进制文件，按照提示输入YYYYMMDD(四位数的年份+月份+日)就可以自动计算输入时间的临时密码。
</br>_**如果这里输入为空，那默认获取本地时间计算临时密码。**_</br>
同时可以配合[computer-room-onekey-killercmd](https://github.com/lzh173/computer-room-onekey-killercmd)这个项目使用，先使用临时密码退出机房管理助手，然后使用项目里的工具脱控。
